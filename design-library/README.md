# Design Library

Structured corpus for curated UI design examples, patterns, and design languages.

## Purpose
This library is intended to support:
- inspiration and retrieval
- pattern extraction
- design-language clustering
- future model training / fine-tuning for a design AI agent

## Layout
- `schema/` — JSON schemas for core entities
- `sources/` — source registry
- `examples/` — captured examples by curation stage
- `assets/` — screenshots, video clips, thumbnails
- `patterns/` — reusable extracted patterns
- `design-languages/` — higher-level style families
- `indexes/` — generated indexes and browsing helpers

See `docs/design-library-spec.md` for the initial spec.
